#### Test Data
The project contains a test data dump file - testdata.csv

The format of the test data is as follows
event type,col1,col2,....

Each line is a comma separated line with the first col specifying the event name.
The rest of the entries in the line are data points for that event.

Examples -
quote,01:00:00.576,AUDUSD,AGG,0.76127,0.76131
deal,01:02:14.860,AUDUSD,30000,0.76111,B
